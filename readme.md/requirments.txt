# Customer Churn Prediction

This project uses machine learning to predict customer churn using a Telco dataset. It includes data preprocessing, model training, evaluation, and insights generation.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib & Seaborn
- Jupyter Notebook

##  Models
- Logistic Regression
- Random Forest
